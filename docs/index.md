# CareRide AI Documentation

## Overview

CareRide AI is an AI-powered mobility assistance platform that helps elderly individuals, persons with disabilities, and patients access safe and accessible transportation.

## Features

* User Authentication
* Helper Management
* Travel Request Management
* Gemini AI Recommendations
* Supabase Integration
* Swagger API Documentation

## AI Workflow

Travel Request → Gemini AI → Helper Ranking → Recommendation → Supabase Storage
