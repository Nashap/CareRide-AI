# Features

CareRide AI provides a comprehensive set of features.

## For Riders
- **Travel Requests:** Create detailed travel requests.
- **Disability Verification:** Securely upload disability certificates.
- **AI Matching:** Receive AI-generated recommendations.

## For Helpers
- **Profile Management:** Manage skills and availability.
- **Ride Assignment:** Receive tailored ride requests.

## Universal Features
- **Secure Authentication:** JWT-based secure authentication.
- **AI Assistant:** Built-in AI chat.
